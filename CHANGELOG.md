# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2023-09-30

### Added
- Initial release of Eigenvalue Calculator
- Inverse iteration method implementation
- GUI for matrix input with visual feedback
- File input support for batch processing
- Support for matrices up to 10x10
- Configurable precision (epsilon)
- Calculation of eigenvalues and eigenvectors

### Features
- Interactive Windows Forms interface
- Upper triangle input for symmetric matrices
- Automatic normalization of eigenvectors
- Hotelling's deflation for multiple eigenvalues
- Input validation for numeric fields
