# Калькулятор собственных значений симметричных матриц

[![C++](https://img.shields.io/badge/C++-CLI-blue.svg)](https://en.wikipedia.org/wiki/C%2B%2B/CLI)
[![Visual Studio](https://img.shields.io/badge/Visual%20Studio-2019+-purple.svg)](https://visualstudio.microsoft.com/)
[![.NET Framework](https://img.shields.io/badge/.NET%20Framework-4.7.2+-green.svg)](https://dotnet.microsoft.com/)

> Десктопное приложение Windows Forms для вычисления собственных значений и собственных векторов симметричных матриц методом **обратных итераций**.

![Скриншот приложения](assets/screenshot.png)

## Возможности

- Вычисление нескольких собственных значений и соответствующих собственных векторов
- Интерактивный графический интерфейс с вводом матрицы в реальном времени
- Поддержка загрузки данных из файла
- Настраиваемая точность вычислений (эпсилон)
- Визуальная индикация для ввода симметричной матрицы
- Возможность экспорта результатов

## Алгоритм

Приложение реализует **метод обратных итераций** (известный также как метод обратных степеней) с дефляцией Хотеллинга:

1. **Степенная итерация**: Итеративное умножение матрицы на вектор для сходимости к доминирующему собственному значению
2. **Отношение Релея**: Вычисление приближения собственного значения
3. **Дефляция**: Удаление найденного собственного значения из матрицы для поиска следующих
4. **Проверка сходимости**: Остановка при изменении между итерациями меньше эпсилон

### Математическая основа

Для симметричной матрицы **A** метод находит собственные значения λ и собственные векторы **v** такие, что:

```
A · v = λ · v
```

Алгоритм использует итерационную формулу:
```
y(k+1) = A · x(k)
x(k+1) = y(k+1) / ||y(k+1)||
λ(k) = (x(k)ᵀ · A · x(k)) / (x(k)ᵀ · x(k))
```

## Сборка из исходников

### С помощью Visual Studio

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/eigenvalue-calculator.git
cd eigenvalue-calculator
```

2. Откройте `EigenvalueCalculator.sln` в Visual Studio

3. Выберите конфигурацию: `Debug` или `Release`

4. Соберите решение: `Ctrl+Shift+B`

5. Запустите: `F5`

### С помощью MSBuild

```bash
msbuild EigenvalueCalculator.sln /p:Configuration=Release /p:Platform="x86"
```

## Использование

### Режим графического интерфейса

1. Запустите приложение
2. Задайте размерность матрицы (n) - максимум 10x10
3. Укажите количество собственных значений для поиска (K)
4. Задайте точность (eps) - по умолчанию 0,00001
5. Введите элементы матрицы в верхнем треугольнике
6. Введите начальный вектор приближения
7. Нажмите **Вычислить**

### Режим ввода из файла

Создайте текстовый файл в следующем формате:

```
n K eps
a11 a12 a13 ... a1n
    a22 a23 ... a2n
        a33 ... a3n
            ...
                ann
x1 x2 x3 ... xn
```

Пример (матрица 3x3):
```
3 3 0,00001
4 1 2
  3 1
    2
1 1 1
```

Затем выберите опцию **"Из файла"** и загрузите ваш файл.

## Входные параметры

| Параметр | Описание | Ограничения |
|----------|----------|-------------|
| n | Размерность матрицы | 1 ≤ n ≤ 10 |
| K | Количество собственных значений | 1 ≤ K ≤ n |
| N | Размерность вектора | N = n |
| eps | Точность | eps > 0 |

## Выходные данные

Программа отображает:
- **Собственные значения**: Список найденных собственных значений (λ₁, λ₂, ..., λₖ)
- **Собственные векторы**: Соответствующие нормализованные собственные векторы
