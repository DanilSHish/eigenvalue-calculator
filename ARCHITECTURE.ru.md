# Обзор архитектуры

## Системное проектирование

Этот документ описывает архитектуру приложения Калькулятор собственных значений.

### Тип приложения
- **Платформа**: Windows Desktop
- **Фреймворк**: .NET Framework 4.6.1+
- **UI-технология**: Windows Forms (C++/CLI)
- **Паттерн архитектуры**: Одноформенное десктопное приложение с событийно-управляемым UI

## Структура проекта

```
src/
├── MyForm.h          # Класс главной формы с реализацией алгоритма
├── MyForm.cpp        # Точка входа приложения (main)
├── MyForm.resx       # Ресурсы Windows Forms (иконки, макеты)
└── EigenvalueCalculator.vcxproj  # Файл проекта
```

## Архитектура классов

### Класс MyForm
Главный класс формы наследуется от `System::Windows::Forms::Form` и содержит:

#### UI-компоненты (автоматически сгенерированы)
- `dataGridView1` - Сетка ввода матрицы (только верхний треугольник)
- `dataGridView2` - Ввод начального вектора
- `numericUpDown1-3` - Ввод параметров (n, K, N)
- `listBox1-2` - Отображение результатов (собственные значения, векторы)
- `radioButton1-2` - Выбор режима ввода
- `menuStrip1` - Меню приложения

#### Основные методы

| Метод | Назначение | Сложность алгоритма |
|-------|------------|---------------------|
| `Matr()` | Основной расчет собственных значений | O(K × N³) |
| `ReadFromFile()` | Файловый ввод/вывод данных матрицы | O(N²) |
| `initDataGridView_1_2()` | Инициализация UI | O(N²) |

## Архитектура алгоритма

### Метод обратных итераций

```
┌─────────────────────────────────────────────────────────┐
│                    ФАЗА ВВОДА                           │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐  │
│  │  Матрица A  │  │  Вектор x   │  │ Параметры       │  │
│  │ (симметр.)  │  │ (начальный) │  │ K, N, эпсилон   │  │
│  └─────────────┘  └─────────────┘  └─────────────────┘  │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│              ВЫЧИСЛИТЕЛЬНЫЙ ДВИГАТЕЛЬ                   │
│  для каждого собственного значения (b = 1 до K):        │
│    ┌──────────────────────────────────────────────┐    │
│    │ 1. Инициализация вектора x₀                   │    │
│    │                                               │    │
│    │ 2. Цикл степенной итерации:                   │    │
│    │    пока |p - f| > эпсилон:                    │    │
│    │      y = A × x                                │    │
│    │      x = y / ||y||                            │    │
│    │      лямбда = (xᵀ·A·x) / (xᵀ·x)               │    │
│    │                                               │    │
│    │ 3. Сохранить (лямбда, x) как собственную пару │    │
│    │                                               │    │
│    │ 4. Дефляция матрицы (Хотеллинг):              │    │
│    │    A = A - лямбда × x × xᵀ                    │    │
│    └──────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                  ФАЗА ВЫВОДА                            │
│  ┌─────────────┐  ┌─────────────┐                       │
│  │ Собств. знач│  │ Собств. вект│                       │
│  │ (лямбда[])  │  │ (v[][])     │                       │
│  └─────────────┘  └─────────────┘                       │
└─────────────────────────────────────────────────────────┘
```

### Размещение в памяти

```
Макет массива 'a' (размер: M×M):
┌──────────────────────────────────────┐
│ a[0]        a[1]        a[2]  ...   │  <- Хранение верхнего треугольника
│ a[1]        a[2]        a[3]        │
│ a[3]        a[4]        a[5]        │
│ ...                                 │
├──────────────────────────────────────┤
│ a[n1]       a[n1+1]    ...          │  <- Начальный вектор
│                                     │
├──────────────────────────────────────┤
│ a[n2]       a[n2+1]    ...          │  <- Результирующий вектор (y)
└──────────────────────────────────────┘

Где:
  n1 = N × (N + 1) / 2  <- Начало хранения вектора
  n2 = n1 + N           <- Начало хранения результата
```

## Поток данных

### Поток ввода через GUI
```
Ввод пользователя → dataGridView → Валидация → массив mas[][] → Matr()
                ↓
                → numericUpDown → Параметры (K, N, eps)
```

### Поток ввода из файла
```
File.txt → StreamReader → Парсинг → AB[][] → Заполнение UI → mas[][] → Matr()
```

### Поток вывода
```
Matr() → XY[][] → String::Format → listBox → Отображение
```

## Критерии сходимости

Итерация останавливается когда:
```
|p - f| < эпсилон
```

Где:
- `p` = max(|xᵢ|) - максимум текущей итерации
- `f` = max(|xᵢ|) - максимум предыдущей итерации
- `эпсилон` - заданная пользователем точность (по умолчанию: 0,00001)

## Ограничения

### Жесткие лимиты
| Параметр | Лимит | Причина |
|----------|-------|---------|
| M (макс. размер) | 10 | Размещение массива в стеке |
| N (размер матрицы) | 1-10 | Ограничения сетки UI |
| K (собств. значения) | 1-N | Требование алгоритма |
| eps (точность) | > 0 | Математическое ограничение |

### Валидация ввода
- Симметричная матрица (только верхний треугольник редактируемый)
- Только числовой ввод (обработчик KeyPress)
- Все поля обязательны перед расчетом

## Обработка ошибок

| Тип ошибки | Обработка |
|------------|-----------|
| Пустые поля ввода | Предупреждение MessageBox |
| Неверный формат файла | Перехват исключения (неявный) |
| Нечисловой ввод | Фильтр KeyPress |
| Выход за границы | Лимиты NumericUpDown |
